# Mihash

New Hash Library

## Usage Example

```python
from Mihash import *

# Encrypt Message
encrypt("abacus",md5,"hex")

# Encrypt File
encrypt("file.jpg",md5,"hex","output.as")

# Decrypt Message
decrypt("13f27b1072bbf7719d0d267b083ff91c",md5,"wordlist.txt")

```
